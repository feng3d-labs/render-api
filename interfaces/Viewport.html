Interface Viewport

视窗。

包含以下限制。

  • x ≥ 0
  • y ≥ 0
  • width ≥ 0
  • height ≥ 0
  • x + width ≤ this.[[attachment_size]].width
  • y + height ≤ this.[[attachment_size]].height
  • 0.0 ≤ minDepth ≤ 1.0
  • 0.0 ≤ maxDepth ≤ 1.0
  • minDepth ≤ maxDepth
